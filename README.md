Milk Delivery Website

Project Description
Milk Delivery Website is a simple web application that allows customers to order fresh milk online. 
Users can log in and place milk delivery orders using a simple order form.

This project demonstrates basic frontend web development using HTML, CSS, and JavaScript.

Features
- Customer Login Page
- Milk Order Form
- Colorful User Interface
- Scrolling Message Banner
- Simple Login Validation using JavaScript

Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub

2. Open the project folder
3. Open the file "index.html" in any web browser
